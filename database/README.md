# Base de datos

Este directorio contiene el esquema y datos de ejemplo para MySQL 8+.

## Archivos
- `schema.sql`: tablas principales, claves foráneas e índices.
- `seed.sql`: datos mínimos para desarrollo.

## Uso
1. Crea una base de datos en MySQL 8+.
2. Ejecuta `schema.sql`.
3. Ejecuta `seed.sql` si quieres datos de ejemplo.
